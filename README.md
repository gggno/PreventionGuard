## 🌄 배경
장기간 지속된 코로나19 상황으로 인해 방역수칙을 잘 지키지 않는 사람들이 많아졌습니다. 이에 방역수칙의 목적성을 강화하고 인식시키기 위해, 예방리스트에 게임을 반영하여 흥미를 유발시키고 이를 통해 사용자들이 슬기로운 코로나19 예방 생활습관을 기를 수 있도록 돕고자 합니다.
